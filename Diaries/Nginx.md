### 常用指令

* start nginx            // 启动
* nginx -s stop          // stop表示立即停止nginx,不保存相关信息
* nginx -s quit          // quit表示正常退出nginx,并保存相关信息
* nginx -s reload        // 重启(因为改变了配置,需要重启)
